프론트엔드

프론트엔드는 사용자에게 시각적으로 보여지는 부분을 의미한다. 프론트엔드쪽에서는 주로 HTML, CSS, 자바스크립트가 쓰이며, 요즘은 페이스북에서 개발한 자바스크립트에서 파생된 싱글 페이지 애플리케이션 형태의 리액트가 많이 사용된다.

백엔드

백엔드는 사용자에게 드러나지 않는 DB 공간(회원가입, 로그인)을 의미한다. 백엔드는 예전의 경우에 PHP가 널리 사용되었지만, 요즘은 Node.js가 주로 사용된다.

컴퓨터 게임 개발자

게임을 개발하는 사람들을 통틀어서 컴퓨터 게임 개발자라고 하는데요. 크게 어떤 게임을 개발할 것인지 전체적인 기획을 하는 게임 기획자, 게임에 대한 것을 그림이나 그래픽으로 그려내고 구현하는 그래픽 디자이너, 게임의 기술적인 부분을 처리하고 기기에 맞게 구현하는 프로그래머로 나뉩니다.

인공지능 엔지니어

인공지능 엔지니어는 데이터를 분석하고 모델을 개발하여 다양한 문제를 해결하는 데 중점을 둔다. 이들은 주로 Python, TensorFlow, PyTorch, 혹은 기타 다양한 머신러닝 프레임워크를 사용한다. 예를 들어, 딥 러닝 기반의 이미지 분류 시스템을 개발할 때에는 TensorFlow와 같은 도구들을 이용하게 된다.

블록체인 엔지니어

블록체인 엔지니어는 분산 원장 기술을 사용하여 투명하고 안전한 시스템을 구축하는 전문가입니다. 블록체인 엔지니어는 주로 스마트 계약과 분산 애플리케이션(dApps)을 설계하고 구현합니다. 이들은 Ethereum, Hyperledger와 같은 블록체인 플랫폼을 사용하며, 특히 스마트 계약을 작성하기 위해 Solidity와 같은 언어를 활용합니다. 최근에는 블록체인의 보안성과 신뢰성을 극대화하기 위해 다양한 암호화 기술과 합의 알고리즘(예: Proof of Work, Proof of Stake)이 많이 사용되고 있습니다.
